# what-CMS
Scriptul what-CMS a fost conceput pentru testarea si scanarea platformelor de tip CMS: Wordpress, Drupal si Joomla!, impotriva vulnerabilitatilor.

## Dependinte
Pentru a putea functiona la parametrii optimi, scriptul are nevoie de urmatoarele 3 utilitare. Descarcati-le in folder-ul what-CMS, dupa ce faceti o clona local a repository-ului.


- https://github.com/OWASP/joomscan
- https://github.com/Dionach/CMSmap
- https://github.com/droope/droopescan
